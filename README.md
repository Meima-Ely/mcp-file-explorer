# Analyseur de Fichiers

## Description
Ce projet est un serveur MCP qui permet à l'IA de lire des fichiers locaux.

## Fonctionnalités
- Lire des fichiers .txt, .csv, .md
- Lister les fichiers d'un dossier

Étudiant en Big Data à Kénitra

## Technologies utilisées
- Node.js
- MCP SDK
- JavaScript
